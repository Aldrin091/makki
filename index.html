My Projects

i Tap Testing Network

Latency tests were performed to identify potential bottlenecks, while connectivity tests verified that all devices could successfully communicate within their respective VLANs and across subnets. Additionally, stress testing was carried out to assess the network's capability to handle high traffic loads under peak conditions, ensuring stability and robustness.

Following the testing phase, detailed project documentation was created to serve as a comprehensive reference for network design, implementation, and maintenance. This included network diagrams, IP addressing schemes, VLAN configurations, routing protocols, and device configurations. The documentation also outlined troubleshooting steps, testing results, and recommendations for future scalability or improvements.

Betz Networks

Designing a comprehensive and logical network topology using tools like Cisco Packet Tracer and GNS3 to simulate and visualize the network infrastructure. The design includes essential networking components such as routers, switches, wireless access points, and workstations to ensure seamless connectivity and efficient communication across all devices.

The topology was carefully structured to meet the organization's requirements, with routers managing inter-network connections, switches providing efficient data transmission within the network, and access points ensuring secure wireless connectivity for mobile devices. Workstations were strategically positioned within the network to optimize performance, minimize latency, and support resource sharing.
